# Mercury Support

Do you need help with Mercury?

Please open a [new issue](https://github.com/mljar/mercury/issues/new) in Mercury GitHub repository. 
We are trying to respond as quickly as possible (Monday to Friday). That's the best way to get help.

Please describe your issue: 
- screenshots help a lot,
- what operating system are you using?
- which Python version are you using?
- please paste the full error log. 

To get full logs please run Mercury with `--verbose`:

```
mercury run --verbose
```
