# Plotly Pie Chart

`Plotly` is a tool for creating charts, which works with Jupyter Notebook. Among its options is the **Pie Chart**. We've got some examples ready to demonstrate their appearance and functionality, so you can see how they work.

- simple pie chart  
- pie chart with `numpy` data
- pie chart with `pandas` data
- an interactive pie chart 

If you need any information about **Plotly** check their docs: [**Plotly Docs**](https://plotly.com/python/).

</Callout>

## Pie Chart 

Example using only `plotly` package:

```python 
# import packages 
import plotly.express as px 

# create data 
x = [20,25,15,40]

# plot 
fig = px.pie(values=x)
fig.show()
```

## Pie Chart with Numpy Data

Display `numpy` data as a pie chart using `plotly`:

```python 
# import packages 
import plotly.express as px 
import numpy as np

# create data 
x = np.array([43,75,89,74,72])

# plot 
fig = px.pie(values=x)
fig.show()
```

<Screenshot src={numpychartPng} alt="Plotly pie chart with numpy data. " />

## Pie Chart with Pandas Data

Make a pie chart with `pandas` data using `plotly`:

```python
# import packages 
import plotly.express as px 
import pandas as pd 

# create data 
df = pd.DataFrame({'values': [76,54,45,14,72]})

# plot 
fig = px.pie(values = df['values'])
fig.show()
```

## Interactive Pie Chart 

Ordinary charts are dull. Let's make an interactive pie chart with `plotly` and `mercury` packages. You can decide if legend will be visible using mercury widgets. In this case, we're using [**Checkbox**](https://runmercury.com/docs/input-widgets/checkbox/).

```python
# import packages
import plotly.express as px 
import mercury as mr
```

```python
# mercury widget
checkbox = mr.Checkbox(label="Legend")
```

```python
# create data 
x = [9,7,8,12]
names = ["Apples", "Bananas", "Oranges", "Grapes"] if checkbox.value else None

# plot
fig = px.pie(values=x, names=names)
fig.show()
```
