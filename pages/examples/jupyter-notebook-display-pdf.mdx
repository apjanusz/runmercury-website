# Display PDF in Jupyter Notebook

## Required packages

You will need to install `mercury` package to run this example.

## Notebook

Let's import required packages:

```python 
import mercury as mr
```
 
Please choose PDF which you want to display and write following code:
```python 
mr.PDF("example.pdf")
```

<Callout type="info">
PDF and notebook have to be in the same directory.
</Callout>

## Mercury App 

Please start Mercury in the same directory as your notebook:

```bash
mercury run
```
