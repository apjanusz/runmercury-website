# Export PDF or HTML

It is possible to export executed notebook to PDF or HTML file. You need to click on `Download` button in the sidebar and select desired format.

<Screenshot src={animation} alt="Export executed notebook to PDF" />

import { Callout } from 'nextra-theme-docs'

<Callout type="info">
  First time PDF download will trigger `pyppeteer` download and installation if not available in the system.
</Callout>

Please make sure you have Chromium installed. It can be done with the following command:

```bash
pyppeteer-install
```
