# Welcome message

When you open Mercury you will see default `Welcome` header. You can easily modify this message. Please add `welcome.md` file in the same directory as your notebooks. 
In this file you can write any text with Markdown syntax. Mercury will check if `welcome.md` file exists and will load your custom welcome message.
