# MultiSelect

Adds multi-select widget (you can select multiple values).

```python
import mercury as mr
```
```python
# add select widget
selected = mr.MultiSelect(label="Please select letters", 
                          value=["a", "b"], 
                          choices=["a", "b", "c", "d", "e"])
```
```python
# access selected values
print(selected.value)
# ["a", "b"]
```

## Parameters

- **value** (_list of strings_) - initial selection.
- **choices** (_list of strings_) - available choices in the widget.
- **label** (_string_) - label near the widget.
- **url_key** (_string_) - set this value if you would like to set `MultiSelect` value with URL paramters.
- **disabled** (_boolean_) - disbale widget in the sidebar, default is set to `False`.
- **hidden** (_boolean_) - hide widget in the sidebar, default is set to `False`.



## Share widget value in URL

Let's create the `MultiSelect` widget with `url_key="selected"`:

```python
import mercury as mr

app = mr.App(title="MultiSelect", description="Demo of MutiSelect widget", show_code=True)

# add widget
selected = mr.MultiSelect(label="Please select letters", 
                          value=["a", "b"], 
                          choices=["a", "b", "c", "d", "e"],
                          url_key="selected")

# print widget value
print(selected.value)
```

You can share the widgets values with other users. Please click the `Share` button in the sidebar bottom. 
It will open a dialog with URL.

The URL will look like:

```
https://your-server-address.com/app/notebook-name?selected=a,b
```

Please note that `?selected=a,b` is added at the end of URL address.
